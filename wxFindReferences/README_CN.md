# wxFindReferences

[English](README.md) | [简体中文](README_CN.md)

wxFindReferences 是一个用于 XIDE 的代码导航工具，允许开发人员快速在当前项目中查找所选标识符的所有引用。

## 功能

- 在活动文件中高亮显示所选标识符。
- 在当前项目的所有文件中搜索引用。
- 在错误/警告(Errors/Warnings)面板中显示结果，便于导航。

## 使用方法

1. 在编辑器中右键单击标识符。
2. 从上下文菜单中选择"查找所有引用"。
3. 在错误/警告(Errors/Warnings)面板中查看结果，并点击以导航到相应的文件和行。
